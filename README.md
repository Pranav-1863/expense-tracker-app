# Expense Tracker App

Personal finance Android app for tracking daily expenses built with Kotlin and Jetpack Compose.

## Features
- Add, view, and delete expenses
- Track expense descriptions and amounts
- Modern UI with Jetpack Compose

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
- MVVM Architecture

## Known Issues
- Some minor bugs currently being fixed

## Installation
Clone the repository and open in Android Studio
